This thread is the rolling feed of project updates. Each release gets a comment here with what shipped + what's next. Subscribe to this discussion (the 🔔 in the top-right of the page) to get notified.

### What you'll see here

- ⚡ **What just shipped** — short blurb per release pointing at the highlights (Migration Wizard v2, Sonos URL fix, Docker image now bundles spotdl, etc.)

- 📊 **Quarterly-ish roadmap recap** — broader strokes than the per-release notes; ties back to the [Roadmap discussion](#)
- 🚨 **Breaking changes / migration steps** — when you'll need to do something on your install (rare but important)
