## 🎉 Show off your Vynl setup

This thread is for sharing how you're running Vynl, what your library looks like, and the weird hacks you did to get it working. Pictures encouraged.

### Some prompts to get you started

- 📊 **Library size** — how many tracks / albums / playlists?
- 🖥️ **Where it runs** — Synology / UGREEN / Unraid / homelab tower / Mac Mini / Raspberry Pi?
- 🔊 **What plays it** — Sonos speakers / headphones / browser / iOS app / something else?
- 📦 **Migrated from?** — Spotify / Apple Music / iTunes library / cold start?
- 🎨 **Favourite feature so far** — the part you didn't know you'd love?
- 🛠️ **A hack you did** — config tweak, integration with another tool, weird workflow?

### Format suggestion

```
## My Vynl setup

**Hardware:** UGREEN DXP4800 Plus, 16 GB RAM, Tailscale mesh

**Library:**
- 14,232 tracks / 1,847 albums / 412 artists
- Migrated from a 12-year Spotify history

**Playback:** Sonos Beam + 2× One SLs in living room

**Favorite so far:** The 5-Vynl Crown ceremony — felt silly until
I rated my first "all time" track and now I check the auto-playlist
every week.

**Quirk:** I bind-mount /mnt/music readonly so beets can scan but
not move files; my Plex library lives in the same directory.

[screenshot]
```
